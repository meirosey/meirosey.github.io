Birdie Mobile Redesign

During the exploration in user experience & interface design, I had heard numbers of Birdie Mobile users complaining the apps are in poor user experience. While referencing user research, I had define the user goal, create user joinery maps, and proposed the following redesign solutions for Birdie Mobile.

Increasing user experience in the data monitor

There are numerous of data monitor applications allow users to use. Different function and features were specially introduced under various purpose. Still, they all share same common spot: Direct, accurate & easy to read information display. However, does that means the same for Birdie Mobile?

About Birdie Mobile

Birdie is a telecommunications provider which avoid all the complex contract procedures. To provide a easy monitor, news updates & customer support for the plan customers. Birdie Mobile is created. From time to time, there will be discount promotion cooperate with different local company. User have to logon and grab the discount. P2P data transfer function is one of the most outstanding feature amount the industry.

The inconvenience data & interactions

As one of the user of Birdie, always hear others annoyed with the messy complex interface of the Birdie Mobile app (ver. 1.74.1).
So, where is the application goes wrong? I have analysis the issues as follows:

Design for a better user experience

How can Birdie Mobile increase a better user experience? By ensuring user complete their real accomplishments.
The redesign was created for experience level rather than a product level.

Understanding different types of users

It’s important to understand the general type of users by looking into to communicate insights and user goals.

Circuits key metrics & user needs

Taking the key metrics which measured by data, I aligned the following user needs and metrics based on the personas.

1 - Easy & direct layout

" I am not familiar where to look what I needs. Even I had been using for server years. "

2 - Clear Notification

" I always missed and want to be reminded when having discount promotions/updates. "

3 - Easy management

" I have to pay for multiple mobile numbers. And I have to arrange them one by one… "

Discover deeper with customers need, points of actions

To understand where the miscommunication between the user and the app, I had created an emotional journey maps of Wincy Li & Kenos Lai, the 2 main personas meets the both user and business needs. The journey map identify behaviour, emotions, and the interaction across the app using. Creating these maps not only will help to visualise the user flow in Birdie Mobile, but also help to communicate with a compelling story within the team.

Recommend solution design 1:

Redesign for interface of the app – For easier & faster navigation

The redesign bottom navigation bar provides the function mostly used by the users. Including the main page, bill information, news, inquiry & notifications.
Animations are also be building for the design to enhance the overall looks and interactions.

Recommend solution design 2:

Native notifications - For better communication & notification with the users

Clear notification & reminders will be shown depends on needs.

3 main cases shall be reminded for users:
1) Bill due date
2) Almost end of data effective dates while users still have
     lot data left
3) New promotion / announcement

With the clear, pop-up notification, will greatly reduce the miss case for the users.

Recommend solution design 3:

Allow for monitor multiple user – Family plans

To enhance the easy management and usability for family party, Family Plans is introduced.

Major function are included:
1) Allow users to monitor data amount used
2) Allow users to monitor and pay and request data form the other family
　member – all by 1 click
3) Allow request multiple user for data in P2P page

These functions allow users to have easy manage all by one-click and displays.

Findings & Learning

Balance of 'Functional' & 'Beautiful'

The original design of Birdie Mobile is functional. Although this is not in the greatest practise, users can still capture information they are looking for. However, it don't satisficed them, not only with the messy but also the non-fancy interface. This have let me deeply impressed the balance of the application's 'functionality' and 'beauty'
